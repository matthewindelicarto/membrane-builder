# Membrane Builder

Build lipid bilayer and TPU membranes, visualize molecular permeability.

## Live Demo

**[Launch Membrane Builder](https://membranebuilder.streamlit.app)** - Free web app, no installation required.

## Features

- **Lipid Membranes**: Build bilayers with POPC, POPE, CHOL, and more
- **TPU Membranes**: Build thermoplastic polyurethane membranes with Sparsa and Carbosil polymers
- **Permeability Prediction**: Animated visualization of molecules passing through membranes
- **MD Simulation Setup**: Generate GROMACS input files for molecular dynamics

## Available Lipids

| Name | Description |
|------|-------------|
| POPC | Phosphatidylcholine |
| POPE | Phosphatidylethanolamine |
| POPG | Phosphatidylglycerol |
| POPS | Phosphatidylserine |
| CHOL | Cholesterol |
| PSM | Sphingomyelin |

## TPU Polymers

| Name | Description |
|------|-------------|
| Sparsa1 | High permeability amphiphilic polyurethane |
| Sparsa2 | Modified Sparsa formulation |
| Carbosil1 | Silicone-polycarbonate polyurethane |
| Carbosil2 | Modified Carbosil formulation |
